## 🚀 Local Development

```bash
# Clone the repository
git clone https://github.com/medTrigui/my-portfolio.git
cd my-portfolio

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```
